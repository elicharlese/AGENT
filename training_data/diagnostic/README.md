# Diagnostic Mode Training Data

## 🔍 Overview

The **Diagnostic Mode** specializes in problem diagnosis, root cause analysis, troubleshooting, and systematic investigation. This mode trains the AGENT LLM to excel in identifying issues, analyzing symptoms, and providing accurate diagnostic assessments.

## 🎯 Core Capabilities

### Problem Diagnosis
- **Symptom Analysis**: Pattern recognition in problem manifestations
- **Root Cause Analysis**: Systematic cause identification methodologies
- **Differential Diagnosis**: Multiple hypothesis evaluation and testing
- **System Health Assessment**: Comprehensive system evaluation

### Troubleshooting
- **Systematic Investigation**: Step-by-step problem resolution
- **Error Analysis**: Log analysis, error pattern recognition
- **Performance Diagnosis**: Bottleneck identification, resource analysis
- **Failure Mode Analysis**: Failure prediction and prevention

### Diagnostic Tools
- **Monitoring Systems**: Real-time system monitoring and alerting
- **Testing Frameworks**: Automated diagnostic testing
- **Analysis Algorithms**: Pattern matching, anomaly detection
- **Reporting Systems**: Diagnostic report generation and visualization
